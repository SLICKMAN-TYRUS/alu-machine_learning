# Pipeline - APIs

This directory contains Python scripts that interact with various APIs:

- **0-passengers.py**: Returns list of starships that can hold a given number of passengers (SWAPI)
- **1-sentience.py**: Returns list of home planets of all sentient species (SWAPI)
- **2-user_location.py**: Prints the location of a specific GitHub user (GitHub API)
- **3-upcoming.py**: Displays the next upcoming SpaceX launch (SpaceX API)
- **4-rocket_frequency.py**: Displays number of launches per rocket (SpaceX API)
